# patt2d_q_sfm-Python
2D BEC optomechanical pattern formation simulation (Python) + plotting routines

A 2D simulation of a BEC interacting with light retroreflected from a single feedback mirror (SFM).

    patt2d_sfm_q.in : simulation parameters

    patt2d_sfm_q.py : simulation code

    snapshots2d_BEC_intens.py : plots snapshots of BEC density and optical intensity at different times.
